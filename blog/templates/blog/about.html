Blog Web Application

This is my first full-featured web application written using Django Python Framework. This is a blog app where most of the common functionalities of a web application can be built such as:

  • Web Fundamentals (HTML/CSS/JS)
  • Bootstrap
  • Jinja
  • Model-View-Template Design Pattern
  • User Registration
  • User Authentication
  • Forgot Password Email Reset Feature (I used Gmail SMTP.)
  • Database
  • Object Relational Mapping
  • CRUD Operations
  • Local File Upload
  • Connect Files to Amazon Web Services, S3 Bucket
  • Deployment (I used Heroku.)
